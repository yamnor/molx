molx

Molecular structures, one link away

View, style, and share molecular structures from public structure files without uploading the structure data to molx.

Paste XYZ, PDB, SDF, MOL2, CIF, or CUBE data locally, open a public file with ?url=..., or register a source URL to create a short molx link.

For usage examples, display controls, and self-hosting notes, see the molx docs.

Created by Nori Yamamoto, a computational chemist based in Japan.
