# Display controls

molx keeps the visible interface small and puts extra display features in the command palette.

Open the command palette from the command icon in the top-right toolbar, or with the keyboard shortcut:

```text
Ctrl+K / Command+K
```

## Style

Available styles:

- Ball & Stick
- Stick
- Sphere
- Line
- Cartoon

PDB files default to Cartoon. Hetero atoms are shown with Ball & Stick when Cartoon is active.

## Color

Available color modes:

- Element
- Chain
- Residue
- Single

## Labels and surfaces

Labels can be disabled or shown by atom or residue. VDW surface display can also be toggled.

## Motion

Rotation and animation can be toggled independently:

- Rotation turns continuous spin on or off.
- Animation plays multi-frame structures when available.

## Save defaults

When you open a registered link with its private edit URL, you can save the current display settings as the default for that short link.

Viewers without the edit URL can still change the display locally, but cannot update the registered default.
